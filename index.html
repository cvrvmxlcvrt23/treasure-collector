Treasure collector

How to play

  • You will be given a random number at the start of the game."Number To Get"
  • There are four icons below.

  • sail boat
  • anchor
  • compass
  • gold

  • Each image will have number value applied to it.
  • By clicking on the image you will add until you reach the "number to get"
  • if you go over that number you will lose.
  • The value of each image is hidden from you until you click on it.
  • Each time when the game starts, the game will change the values of each image.
